# 🧠 Tamashii AI — Emotional Memory Chatbot (v6.0 Final)

Tamashii AI is an intelligent, emotional, memory-powered chatbot built with Python and OpenAI's GPT API.  
It understands the user's emotion, remembers past conversations, and speaks with human-like voice.

---

## 🌟 Features

- 🤖 ChatGPT-powered intelligent brain
- 🧠 Emotional classification (detects if you're happy, sad, etc.)
- 💾 Memory system that remembers recent interactions
- 🗣️ Text-to-Speech using `pyttsx3`
- 🪟 Simple GUI using `tkinter`
- 📄 Chat log with timestamp, emotions, and memory
